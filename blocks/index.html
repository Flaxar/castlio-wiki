Storage blocks

Locked chest info: You will need a key to open locked chest. Using that key will bind the key to the chest, chest will open only using that key. One key can be used to bind multiple locked chests, but locked chest has only one binding key. Do not lose your keys.

Walls

Locked doors work in the similar manner as locked chests. Key is crafted also the same. You can even use one key for your chests and doors.
